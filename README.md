# 🌡 Temperature Converter Web App

## 📌 Project Title

**OIBSIP Level 1 - Task 3**
Temperature Converter (OIBSIP1-3)

---

## 📖 Description

This is a simple and interactive **Temperature Converter Web Application** built using **HTML, CSS, and JavaScript**.
It allows users to input a temperature value and convert it between **Celsius, Fahrenheit, and Kelvin**.

The application includes input validation and provides real-time conversion results with a clean and modern UI.

---

## 🚀 Features

* Input field for entering temperature
* Dropdown selection for unit conversion (Celsius, Fahrenheit, Kelvin)
* Swap button to quickly switch units
* Convert button to trigger calculation
* Displays result with correct unit
* Input validation (prevents invalid entries)
* Responsive and modern UI design

---

## 🛠 Technologies Used

* HTML5
* CSS3 (Glassmorphism UI + Animations)
* JavaScript (DOM manipulation & logic)

---

## 📐 Conversion Logic

The conversion is done in two steps:

1. Convert the input temperature to **Celsius**
2. Convert from **Celsius to the target unit**

---

## 📷 UI Components

* Temperature Input Field
* Unit Selection Dropdowns (From & To)
* Swap Units Button
* Convert Button
* Result Display Section

---

## ⚠ Validation

* Ensures input is not empty
* Ensures input is a valid number
* Displays warning message for invalid input

---

## 🎯 How to Use

1. Enter a temperature value
2. Select the unit you are converting **from**
3. Select the unit you want to convert **to**
4. Click the **Convert** button
5. View the result instantly
